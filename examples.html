Example 1: Simple Schema Export

Use Case: Export a single schema

expdp scott/tiger directory=DATA_PUMP_DIR dumpfile=scott.dmp schemas=scott

Key Parameters:

  • directory: Directory object for files
  • dumpfile: Output dump file name
  • schemas: Schema to export

Example 2: Multiple Schemas with Compression

Use Case: Export multiple schemas with compression

expdp system/password directory=DATA_PUMP_DIR dumpfile=multi_schema.dmp schemas=scott,hr,oe compression=all parallel=4

Key Parameters:

  • schemas: Multiple schemas separated by comma
  • compression: Compression type (ALL, DATA_ONLY, METADATA_ONLY)
  • parallel: Number of parallel processes

Example 3: Table Export with Query Filter

Use Case: Export specific tables with data filter

expdp scott/tiger directory=DATA_PUMP_DIR dumpfile=emp_dept.dmp tables=scott.emp,scott.dept query=scott.emp:"WHERE deptno=10"

Key Parameters:

  • tables: Specific tables to export
  • query: WHERE clause to filter data

Example 4: Schema Import with Remap

Use Case: Import schema to different schema name

impdp system/password directory=DATA_PUMP_DIR dumpfile=scott.dmp remap_schema=scott:scott_dev table_exists_action=replace

Key Parameters:

  • remap_schema: Map source schema to target schema
  • table_exists_action: Action when table exists (SKIP/APPEND/TRUNCATE/REPLACE)

Example 5: Metadata Only Export

Use Case: Export only DDL without data

expdp scott/tiger directory=DATA_PUMP_DIR dumpfile=scott_ddl.dmp schemas=scott content=metadata_only

Key Parameters:

  • content: METADATA_ONLY exports only DDL

Example 6: Full Database Export

Use Case: Export entire database

expdp system/password directory=DATA_PUMP_DIR dumpfile=full_db_%U.dmp full=y parallel=8 filesize=2G

Key Parameters:

  • full: Full database export
  • %U: Placeholder for multiple files
  • filesize: Maximum size per file

Example 7: Exclude/Include Objects

Use Case: Export schema excluding statistics

expdp scott/tiger directory=DATA_PUMP_DIR dumpfile=scott.dmp schemas=scott exclude=statistics

Key Parameters:

  • exclude: Exclude specific object types
  • include: Include only specific object types

Example 8: Tablespace Export

Use Case: Export all objects in a tablespace

expdp system/password directory=DATA_PUMP_DIR dumpfile=users_ts.dmp tablespaces=users

Key Parameters:

  • tablespaces: Export all objects in specified tablespace(s)

Example 9: Import with Tablespace Remap

Use Case: Import to different tablespace

impdp system/password directory=DATA_PUMP_DIR dumpfile=scott.dmp remap_tablespace=users:users2 remap_schema=scott:scott_test

Key Parameters:

  • remap_tablespace: Map source tablespace to target
  • remap_schema: Can combine with schema remap

Example 10: Network Import (Direct Database Link)

Use Case: Import directly from remote database

impdp system/password directory=DATA_PUMP_DIR network_link=source_db schemas=scott remap_schema=scott:scott_copy

Key Parameters:

  • network_link: Database link to source database
  • No dumpfile needed - direct transfer
