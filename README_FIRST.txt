WHAT YOU HAVE:
--------------
A complete web application that converts Oracle Data Pump command line 
syntax (expdp/impdp) to DBMS_DATAPUMP API PL/SQL scripts.

QUICK START:
-----------
1. Open "index.html" in your web browser
2. Enter your expdp/impdp command OR use the form
3. Click generate button
4. Copy the PL/SQL script
5. Run it in your Oracle database

DEPLOY TO WEB (5 MINUTES):
--------------------------
1. Push code to GitHub
2. Enable GitHub Pages in repository Settings
3. Access at: https://YOUR_USERNAME.github.io/orace-datapump/
4. Share with your team
